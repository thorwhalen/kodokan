kodokan.config

Where kodokan keeps its data (downloaded clips + derived artifacts).

Large/churny artifacts (videos, per-frame keypoints, rendered overlays) live outside the repo by default — under ~/kodokan_data — so they don’t bloat the (Dropbox-synced) source tree. Override the root with the KODOKAN_DATA_DIR environment variable.

kodokan.config.KODOKAN_DATA_ENV = 'KODOKAN_DATA_DIR'

Env var overriding the data root (default ~/kodokan_data).
