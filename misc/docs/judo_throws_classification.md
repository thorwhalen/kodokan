# Official Kodokan Judo Throws & Terminology Analysis

This document contains a comprehensive linguistic and mechanical breakdown of the 68 official Kodokan Judo throws (*Nage-waza*). It includes:
1. An enhanced Japanese word frequency table with nuanced conceptual translations in both English and French.
2. A complete classification table grouped by official technical categories with French designations recognized by the *Fédération Française de Judo* (FFJudo).
